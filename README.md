# NoEgo (TFE) — Site statique (Vercel)

Site 4 pages (Accueil / About / Shop / Team) avec thème Noël (guirlande + neige).

## Lancer en local
- Ouvre `index.html` directement
- ou utilise **Live Server** (VS Code)

## Déployer sur Vercel (via GitHub)
1. Crée un repo GitHub et push ce dossier.
2. Sur Vercel: **New Project** → importe le repo.
3. Framework preset: **Other** (site statique)
